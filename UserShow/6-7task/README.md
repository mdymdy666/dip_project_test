# 6-7 身份证鲁棒性测试交付包

更新时间：2026-06-07 18:41

## 本目录用途

本目录集中整理 2026-06-07 本轮任务的全部可交付资料，用于向协作者或老师说明：项目对 `datas` 文件夹下身份证图片进行了鲁棒性测试，并将测试结果追加到现有课程答辩 PPT 后部。

本目录只聚焦身份证识别鲁棒性测试，不重新整理车牌识别内容。所有统计均来自实际脚本运行结果，没有将失败样例美化为成功。

## 核心结论

- 测试样本总数：22 个。
- `datas` 原始样本：6 个。
- 基于 `datas/normal.png` 生成的补充扰动样本：16 个。
- 定位成功：21/22。
- 身份证号码 OCR 完全正确：11/22。
- 失败样例已保留验证截图，用于说明倾斜、噪声、低分辨率和遮挡等边界问题。

## 目录索引

- `source_datas/`：本轮使用的 `datas` 原始身份证样本与样本总览图。
- `robustness_generated/`：补充生成的鲁棒性扰动样本，包含亮度、对比度、噪声、模糊、旋转、缩放、透视、遮挡、裁剪、JPEG 压缩和组合扰动。
- `verification_screenshots/`：每个测试样本的运行验证截图，包含输入图、灰度化、二值化、形态学候选区域和识别输出。
- `processed_steps/`：逐样本图像处理中间步骤，便于复核灰度化、二值化、形态学和候选区域定位。
- `tables/`：鲁棒性测试汇总表、逐样本明细、JSON 结果和文件清单。
- `ppt_assets/`：用于 PPT 的统计图、汇总表截图、成功案例和失败案例图。
- `ppt_slides/`：追加到 PPT 后部的 9 页页面源图。
- `ppt/`：最终 PPT 和追加前的源 PPT 备份。
- `scripts/`：本轮测试与 PPT 追加脚本。
- `docs/`：原始测试说明、PPT 更新说明和自检记录。
- `logs/`：本轮日志摘录。

## 推荐查看顺序

1. 先看 `tables/idcard_robustness_summary_template.csv`，这是按用户给定模板整理的汇总表。
2. 再看 `ppt_slides/`，可快速了解 PPT 后部新增章节。
3. 如需核验证据，打开 `verification_screenshots/` 和 `processed_steps/`。
4. 最后打开 `ppt/身份证车牌识别数字图像处理答辩_身份证鲁棒性测试版.pptx` 查看完整答辩文件。
